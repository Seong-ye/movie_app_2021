# Movie App 2021
---

### 개요
* 토렌트 영화 웹 순위 어플리케이션

### 특징
* HashRoute를 이용해 페이지 Route
* PropType을 활용해 데이터 검사
* yts Api를 통해 영화 데이터로 활용

### 적용 기술
* HTML5
* CSS3
* JavaScript
* React js
* Axios
